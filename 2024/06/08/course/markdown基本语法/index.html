Markdown 基本语法学习

Markdown是一种轻量级的标记语言，使用简单的文本格式来编写文档，可以很容易地转换为HTML等格式。

1. 标题

使用#表示标题，#的数量表示标题的级别。

2. 段落

段落之间需要有一个空行分隔。

3. 强调

使用* 或者 _包围文字来表示斜体
使用 ** 或 __ 包围文字来表示粗体

4. 列表

无序列表

使用 *、- 或 + 表示无序列表项。

有序列表

使用数字加上 . 表示有序列表项。

5. 链接

使用 链接文本 来创建链接。

6. 图片

使用 来插入图片。

7. 引用

使用 > 表示引用。

8. 代码

行内代码

使用反引号 ``` 包围行内代码。

代码块

使用三个反引号 或四个空格表示代码块。

9. 表格

使用 | 分隔表格的列，使用 - 创建表头。

10. 分割线

使用三个或更多的 *、- 或 _ 创建分隔线。

11. 转义字符

使用反斜杠 \ 转义特殊字符。
